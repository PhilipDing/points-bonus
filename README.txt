寒假积分大放送 - 配置说明
=========================

配置文件说明：
-------------

1. tasks.txt - 任务列表配置
   - 格式：JSON 数组
   - 每个任务包含 name（任务名称）和 points（积分值）
   - 示例：
     [
       {
         "name": "完成数学作业 📚",
         "points": 10
       }
     ]

2. rewards.txt - 积分兑换配置
   - 格式：JSON 数组
   - 每个奖励包含 name（奖励名称）和 points（所需积分）
   - 示例：
     [
       {
         "name": "小零食 🍬",
         "points": 20
       }
     ]

使用说明：
---------

1. 直接编辑 tasks.txt 和 rewards.txt 文件
2. 保存文件后刷新浏览器页面
3. 新的配置会自动加载

注意事项：
--------

- 请保持 JSON 格式正确
- 每个字段的值要用双引号包裹
- 数组最后一个元素后面不要有逗号
- 积分值为正整数

功能说明：
--------

- 每日签到：获得 -5、0、5、10 积分（随机）
- 任务列表：每天早上7点自动重置
- 积分兑换：根据配置文件显示可兑换商品
- 积分记录：记录所有积分获取和使用情况
